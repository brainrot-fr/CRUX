## Introduction

CRUX is a self-learning resource for someone who is targeting IT or any other programming related career like Python developer, Javascript developer, JAVA developer, etc,.

## Purpose of CRUX

CRUX was made with the sole intention of lillahiyat to help out my mahdavi brothers/sisters with whatever I have learned/understood so far.

## Who is CRUX for?

Mahdavis who are willing to:

- Get Familiar with Computers, 
- Learn Programming,
- Learn Linux,
- Get into IT career,etc,.
- Know how to use Artificial Intelligence better than others.

## Prerequisites for CRUX

1. Understand English.
2. Have a desktop/laptop.
3. Have an Internet Connection to install and setup the development environment.
4. Have the dedication to finish the course.

> If you choose to install linux alongside windows on your computer, you need a USB flash drive of not less than 8 GB (not 8 Gb) capacity (8-GB works but 16 GB is recommended)
